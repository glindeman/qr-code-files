# QR Code Generation Instructions

1. Upload file to "files" directory
2. Copy this link: https://glindeman.github.io/qr-code-files/files/{file_name}
3. Replace {file_name} with the name of the file, including the extension
4. Go to https://www.qrcodechimp.com/
5. Add the previous URL and save the newly-created QR code
6. Download the QR code and share!
